# Qubick-Editor
Qubick软件的可视化编辑器部分

## 项目说明
一个基于浏览器的自动化流程编辑器，具有现代化的磨砂玻璃风格界面。支持模块化拖拽编程，可导出多种编程语言。

## 已实现功能
- [x] 现代化磨砂玻璃风格界面
- [x] 多种界面主题（磨砂玻璃、护眼模式）
- [x] 自定义背景（必应壁纸、本地上传、URL）
- [x] 模块拖拽功能
- [x] 多标签页编辑
- [x] 动态加载模块配置
- [x] JSON配置导入导出

## 功能特性
通过拖拽生成任何你想生成的，包括且不限于：
- 演示文稿 (Slidev, Marp, etc.)
- 自动化脚本 (Python, JavaScript, etc.)
- 儿童编程，嵌入式开发 (Scratch, MicroPython, etc.)
- 视频 (Manim)
- 神经网络 (TensorFlow, PyTorch, etc.)
- 一个算法，多种编程语言实现导出 (Python, C, Java, etc.)
- 个人主页生成 (Hugo, Jekyll, etc.)
- 个人简历生成
- Project Page 生成
- 应用开发 (Streamlit, Electron, etc.)

## 界面组件
- 左侧边栏：模块列表和分类

## 使用说明

### 界面操作
- 左侧选择模块
- 拖拽模块到编辑区
- 双击模块打开编辑
- 配置参数保存

### 设置说明
- 点击左下角设置图标
- 选择界面模式
- 调整界面参数
- 设置背景图片

### 数据管理
- 使用导入按钮加载配置
- 使用导出按钮保存配置

## 模块编写指南
对于模块的编写，我们使用基于可视化编辑器进行的方式。Qubick-Editor中一切都将是模块(积木)，支持导出多种编程语言。

### 模块类型
1. 函数式编程模块
   - 传入变量
   - 传出变量
   - 自动识别变量类型

2. 面向对象编程模块
   - 新建实例
   - 调用实例
   - 自动识别属性和方法

### 基础模块
- 顺序结构
- 循环结构
- 判断结构

### 自动化功能
- 自动识别代码中的变量
- 区分输入、输出、中间变量
- 识别类的属性和方法
- AI辅助多语言适配

## 优势
通过Qubick-Editor，节省时间，提高体系化思考能力，关注要做的事情本身即可。在CHATGPT时代，构建框架的能力和体系化思维将成为核心竞争力。
